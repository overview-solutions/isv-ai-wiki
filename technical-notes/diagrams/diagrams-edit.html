Metering topology SVG

diagrams.net cannot load this SVG from a URL automatically. The wiki SVG is plain vector markup (not a native .drawio file). Use the steps below — it takes about 10 seconds.

Quick start

  1. Open diagrams.net (button below).
  2. File → Open from → Device and choose the SVG you saved, or drag the SVG file from Finder onto the canvas.
  3. Edit, then File → Export as → SVG and replace metering-topology.svg in the repo.
