How to edit the topology diagram

Metering & comms network map · SVG source file

What is SVG? SVG (Scalable Vector Graphics) is an open XML format for diagrams. Boxes, lines, and labels are plain text coordinates — not a PNG. Zoom without blur; edit in code or visual tools.

1. Open the diagram file

Repo path (edit in Cursor or clone from GitHub):

isv-ai-wiki/technical-notes/diagrams/metering-topology.svg

View SVG in browser Preview in wiki

2. Edit visually in diagrams.net (easiest)

diagrams.net can open the SVG directly when it is hosted on the web:

Edit in diagrams.net (SVG loaded) Inkscape (desktop)

Or open manually: File → Open from → Device → select metering-topology.svg (required for local-only drafts).

  1. Move boxes, add arrows, change labels
  2. File → Export as → SVG (keep proposed-only classes if possible)
  3. Replace the file in the repo and refresh the wiki

3. Edit in Cursor (small text tweaks)

Open the .svg file. Each box is a <rect>, each label a <text>, each arrow a <line> or <path>.

Install the SVG Preview extension for a side-by-side preview. Use ASCII-only punctuation in labels (no smart dashes).

4. Preview locally

From the isv-ai-wiki folder:

./preview.sh

Then open http://localhost:8765/index.html#notes/metering-2026-05-28

Proposed relay toggle

Green “proposed” paths use class="proposed-only" on proposed-edges and phone-relay. Keep that class if you re-export from draw.io.
