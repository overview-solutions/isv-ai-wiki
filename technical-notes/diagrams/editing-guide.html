How to edit the topology diagram

Metering & comms network map · SVG source file

What is SVG? SVG (Scalable Vector Graphics) is an open XML format for diagrams. Boxes, lines, and labels are plain text coordinates — not a PNG. Zoom without blur; edit in code or visual tools.

1. Edit visually in diagrams.net

diagrams.net (free, browser or desktop):

1. Download SVG 2. Open diagrams.net
Use Import From, not Open From. Open From shows “Not a diagram file” for this SVG.
  1. Download metering-topology.svg (button above).
  2. Open diagrams.net in a new tab.
  3. File → Import From… → Device and select the downloaded file.
  4. Edit boxes, arrows, and labels.
  5. File → Export as → SVG. Replace the file in the repo and refresh the wiki.

Or drag the SVG file from Finder onto the canvas.

2. Edit in Cursor (small text tweaks)

Repo path: isv-ai-wiki/technical-notes/diagrams/metering-topology.svg

Open the .svg file. Each box is a <rect>, each label a <text>, each arrow a <line> or <path>.

Install the SVG Preview extension for a side-by-side preview. Use ASCII-only punctuation in labels (no smart dashes).
