Field notes

  • When comms go down, meters still consume remaining prepaid tokens locally.
  • CIU holds stats the cloud wants but cannot reach when DCU fails.
  • Mesh telecom is already deployed — need supplemental relay, not replacement.
  • Per-village coverage map + drone/satellite 3D survey for repeater placement.
