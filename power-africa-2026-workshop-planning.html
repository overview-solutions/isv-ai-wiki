Workshop session summary

The PowerAfrica hands-on track is structured in three sessions: MAKE (Monday morning, 4 hours) — assemble pole-gateway kits and bench hardware; CODE (Tuesday evening, 2 hours) — firmware, MQTT/northbound profiles, and lab exercises on the built gear; INTEGRATE (Saturday morning, 2 hours) — connect meters, gateway, and operator-facing stack into an end-to-end village metering path.

Planning checklist

  • Kits — have enough for 20; workshop use only, not for sale
  • Cabinets — 3 cabinets to ship / set up in the room
  • Budget — determine total cost including duties, shipping, and kit procurement (under $1,000 target)
  • Room arrangement — how should the workshop space be laid out for hands-on work?
  • Materials — what needs to be prepped before the conference? Power: 5 V for MeshEMS temp reference + SSRs (buck/boost from 18650); CircuitSetup on 3.3 V; see 25 Jun notes · pole power rails
  • Shipping address — get delivery address from Albert in Nairobi
  • Registration — groups joining a workshop session register for the conference day rate on the day they attend (MAKE Mon, CODE Tue eve, INTEGRATE Sat)
  • Non-ISV participants — same day-rate registration; confirm each attendee is registered for the session date they plan to join
